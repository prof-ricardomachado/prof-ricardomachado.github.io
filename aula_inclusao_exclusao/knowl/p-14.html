Note que
\begin{equation*} \#A = \#(A_6\cup A_{15}\cup A_{21}). \end{equation*}
O número de múltiplos de um número natural \(n\text{,}\) entre 1 e 1000, inclusive, é dado pelo quociente da divisão Euclideana de 1000 por \(n\text{,}\) pois, se \(1000 = n\cdot q + r\text{,}\) com \(0\leq r \lt n\text{,}\) então os números \(1\cdot n, 2\cdot n, \ldots, q\cdot n\) são múltiplos de \(n\) e estão entre 1 e 1000. Porém, no exemplo, a contagem dos múltiplos deve ser entre 100 e 1000, inclusive. Para contornar esta situação, definimos os conjuntos
\begin{align*} A_6' \amp = \{n\in\mathbb{N} \mid 1\leq n\leq 99 \text{ e } n \text{ é múltiplo de 6}\};\\ A_{15}' \amp = \{n\in\mathbb{N} \mid 1\leq n\leq 99 \text{ e } n \text{ é múltiplo de 15}\};\\ A_{21}' \amp = \{n\in\mathbb{N} \mid 1\leq n\leq 99 \text{ e } n \text{ é múltiplo de 21}\}. \end{align*}
e os conjuntos
\begin{align*} A_6'' \amp = \{n\in\mathbb{N} \mid 1\leq n\leq 1000 \text{ e } n \text{ é múltiplo de 6}\};\\ A_{15}'' \amp = \{n\in\mathbb{N} \mid 1\leq n\leq 1000 \text{ e } n \text{ é múltiplo de 15}\};\\ A_{21}'' \amp = \{n\in\mathbb{N} \mid 1\leq n\leq 1000 \text{ e } n \text{ é múltiplo de 21}\}. \end{align*}
