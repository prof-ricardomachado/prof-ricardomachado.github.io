Solução 1.9.1.

Inicialmente defina os conjuntos
\begin{align*} A_6 \amp = \{n\in\mathbb{N} \mid 100\leq n\leq 1000 \text{ e } n \text{ é múltiplo de 6}\};\\ A_{15} \amp = \{n\in\mathbb{N} \mid 100\leq n\leq 1000 \text{ e } n \text{ é múltiplo de 15}\};\\ A_{21} \amp = \{n\in\mathbb{N} \mid 100\leq n\leq 1000 \text{ e } n \text{ é múltiplo de 21}\}. \end{align*}
Note que
\begin{equation*} \#A = \#(A_6\cup A_{15}\cup A_{21}). \end{equation*}
O número de múltiplos de um número natural \(n\text{,}\) entre 1 e 1000, inclusive, é dado pelo quociente da divisão Euclideana de 1000 por \(n\text{,}\) pois, se \(1000 = n\cdot q + r\text{,}\) com \(0\leq r \lt n\text{,}\) então os números \(1\cdot n, 2\cdot n, \ldots, q\cdot n\) são múltiplos de \(n\) e estão entre 1 e 1000. Porém, no exemplo, a contagem dos múltiplos deve ser entre 100 e 1000, inclusive. Para contornar esta situação, definimos os conjuntos
\begin{align*} A_6' \amp = \{n\in\mathbb{N} \mid 1\leq n\leq 99 \text{ e } n \text{ é múltiplo de 6}\};\\ A_{15}' \amp = \{n\in\mathbb{N} \mid 1\leq n\leq 99 \text{ e } n \text{ é múltiplo de 15}\};\\ A_{21}' \amp = \{n\in\mathbb{N} \mid 1\leq n\leq 99 \text{ e } n \text{ é múltiplo de 21}\}. \end{align*}
e os conjuntos
\begin{align*} A_6'' \amp = \{n\in\mathbb{N} \mid 1\leq n\leq 1000 \text{ e } n \text{ é múltiplo de 6}\};\\ A_{15}'' \amp = \{n\in\mathbb{N} \mid 1\leq n\leq 1000 \text{ e } n \text{ é múltiplo de 15}\};\\ A_{21}'' \amp = \{n\in\mathbb{N} \mid 1\leq n\leq 1000 \text{ e } n \text{ é múltiplo de 21}\}. \end{align*}
Assim,
\begin{align*} \#A_i \amp = \#A_i''-\#A_i',\\ \#(A_i\cap A_j) \amp = \#(A_i''\cap A_j'') - \#(A_i'\cap A_j'),\\ \#(A_i\cap A_j\cap A_k) \amp = \#(A_i''\cap A_j'' \cap A_k'') - \#(A_i'\cap A_j'\cap A_k'), \end{align*}
Portanto,
\begin{equation*} \#A_6 = \#A_6'' - \#A_6' = 166-16 = 150, \end{equation*}
pois \(1000 = 6\times 166 + 4\) e \(99 = 6\times 16 + 3\text{.}\)
Analogamente,
\begin{align*} \#A_{15} \amp = \#A_{15}''-\#A_{15}' = 66 - 6 = 60,\\ \#A_{21} \amp = \#A_{21}''-\#A_{21}' = 47 - 4 = 43. \end{align*}
Para o caso das interseções \(A_{n_1}\cap A_{n_2}\cap \cdots \cap A_{n_k}\text{,}\) devemos usar o quociente da divisão Euclideana pelo \(\mathrm{MMC}(n_1, n_2, \ldots, n_k)\text{.}\) Pois, o MMC é o menor múltiplo desses números, por definição. Logo,
\begin{equation*} \#(A_6\cap A_{15}) = \#(A_6''\cap A_{15}'') - \#(A_6'\cap A_{15}') = 33-3 = 30, \end{equation*}
pois \(\mathrm{MMC}(6,15) = 30\text{,}\) \(1000 = 30\cdot 33 + 10\) e \(99 = 30\cdot 3+9\text{.}\) Analogamente,
\begin{align*} \#(A_6\cap A_{21}) \amp = \#(A_6''\cap A_{21}'') - \#(A_6'\cap A_{21}') = 23 -1= 21,\\ \#(A_{15}\cap A_{21}) \amp = \#(A_{15}''\cap A_{21}'') - \#(A_{15}'\cap A_{21}') = 9-0 = 9. \end{align*}
Na interseção dos três conjuntos temos,
\begin{equation*} \#(A_6\cap A_{15}\cap A_{21}) = \#(A_6''\cap A_{15}''\cap A_{21}'')-\#(A_6'\cap A_{15}'\cap A_{21}') = 4-0= 4, \end{equation*}
pois \(\mathrm{MMC}(6,15,21) = 210\text{,}\) \(1000 = 210\cdot 4 + 160\) e \(99 = 210\cdot 0+99\text{.}\)
Finalmente, pelo Princípio da Inclusão-Exclusão:
\begin{equation*} \#A = \#(A_6 \cup A_{15} \cup A_{21}) = 150+60+43-30-21-9+4 = 197. \end{equation*}
