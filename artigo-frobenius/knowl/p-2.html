Destacamos dentre as ferramentas computacionais disponibilizadas:

  1. O Algoritmo de Euclides para o cálculo de MDC;
  2. A solução geral para uma equação diofantina linear (em qualquer dimensão).
  3. Gráfico com algumas soluções de Equações Diofantinas Lineares de dimensão 2.
  4. A solução do número de Frobenius em dimensão 2.

Estas tecnologias podem auxiliar bastante o estudo no tema das Equações Diofantinas uma vez que permitem não apenas rápidas soluções para o problema, mas também processos intermediários detalhadamente escritos.
