Seção 2 Reticulado

Chamaremos o plano de coordenadas inteiras de um reticulado.

Subseção 2.1 Caminhos no Reticulado

Observação 2.1.
  1. Escrevemos L para indicar o movimento \((n,m)\to (n+1,m)\text{;}\)
  2. Escrevemos N para indicar o movimento \((n,m)\to (n,m+1)\text{.}\)
Tecnologia 2.2.

Digite uma sequência de L's e N's e clique no botão "Update" para obter o respectivo caminho no reticulado.

Subseção 2.2 Caminhos de Delannoy

Observação 2.4.
  1. Escrevemos L para indicar o movimento \((n,m)\to (n+1,m)\text{;}\)
  2. Escrevemos N para indicar o movimento \((n,m)\to (n,m+1)\text{.}\)
  3. Escrevemos D para indicar o movimento \((n,m)\to (n+1,m+1)\text{.}\)
Tecnologia 2.5.

Digite uma sequência de L's, N's e D's e clique no botão "Update" para obter o respectivo caminho no reticulado.

Tecnologia 2.7.

Digite um ponto \((n,k)\) e clique no botão "Update", para obter o número de caminhos de Delannoy até este ponto e um caminho de Delannoy, gerado aleatoriamente.
