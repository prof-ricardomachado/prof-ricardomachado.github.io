Solução 1.4.9.1.

item a. Para o primeiro grupo, temos um total de \(C_{12, 5}\) maneiras de selecionar as 5 pessoas.

Para o segundo grupo, como 5 pessoas já foram escolhidas, temos 7 pessoas para escolher 4, no total são \(C_{7, 4} \) maneiras de escolher essas pessoas.

Para o terceiro grupo temos um total de \(C_{3,3} \) maneiras de selecionar as 3 pessoas. Portanto a resposta é

\begin{equation*} C_{12,5}\times C_{7,4}\times C_{3,3} = 27720. \end{equation*}

item b. Podemos aplicar a mesma ideia do item anterior, só que os grupos são todos com quatro pessoas. Logo, a resposta é dada por

\begin{equation*} C_{12,4}\times C_{8,4}\times C_{4,4}=34650 . \end{equation*}

item c. Podemos aplicar a mesma ideia do item b., porém, a ordem dos grupos não importa. Logo, precisamos dividir pelo número de maneiras de ordenar os grupos. Portanto, a resposta é

\begin{equation*} \frac{C_{12,4}\times C_{8,4}\times C_{4,4}}{3!} = 5775. \end{equation*}
