Observe que se a cada passo de \((m,n)\) para \((m+1, n+1)\) associarmos uma letra "L" e a cada passo de \((m,n)\) para \((m+1, n-1)\) associarmos uma letra "N", temos uma associação dos caminhos de Dyck com os anagramas do Exemplo 2.6.18 que acabamos de ver. O qual tem sua cardinalidade determinada pelos números de Catalan.
