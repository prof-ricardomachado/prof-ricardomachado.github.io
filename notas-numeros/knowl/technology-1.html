Tecnologia 3.2.10.

Digite uma equação ou inequação e clique fora do espaço de digitação, para tentar obter o conjunto solução usando uma implementação em SageMath.

Observação 3.2.11.
  • Para o valor absoluto \(|x|\text{,}\) use \(abs(x)\)
  • Para escrever \(2x\text{,}\) use \(2*x\)
  • Para igual, use \(==\)
  • Para menor ou igual, use \(\lt=\)
