Parágrafo

Para cada \(y\in Y\) escolha um \(x\in X\) tal que \(f(x)=y\text{,}\) isto é possível, pois \(f\) é sobrejetiva. Defina uma função \(g:Y\rightarrow X\) tal que \(g(y)=x\text{,}\) para cada \(x\in X\) com \(f(x)=y\text{.}\) Então, \(f(g(y))=y\) para todo \(y\in Y\text{.}\) Segue daí que \(g\) é injetiva, pois se \(g(y_1)= g(y_2)\text{,}\) então

\begin{equation*} y_1 = f(g(y_1)) = f(g(y_1))=y_2\text{.} \end{equation*}

Pelo Corolário 5.3.3, \(Y\) é enumerável.
