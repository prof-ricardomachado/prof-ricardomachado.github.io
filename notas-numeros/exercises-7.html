Exercícios 7.4 Exercícios

1.

Verifique as seguintes igualdades:

  1. \(\displaystyle \sqrt{2}-i-i(1-i\sqrt{2})=-2i\)
  2. \(\displaystyle (2-3i)(-2+i)=-1+8i\)
  3. \(\displaystyle \dfrac{5}{(1-i)(2-i)(3-i)}=\dfrac{1}{2}i\)
  4. \(\displaystyle \dfrac{1+2i}{3+4i}+\dfrac{2-i}{5i}=\dfrac{6-8i}{25}\)
  5. \(\displaystyle \overline{\overline{z}+3i}=z-3i\)
  6. \(\displaystyle \dfrac{(\overline{2+i})^2}{3-4i}=1\)
2.

Escreva as expressões abaixo na forma \(a+bi\text{.}\)

  1. \(\displaystyle (4-i)+i-(6+3i)i\)
  2. \(\displaystyle (7+4i)(2-3i)+(6-i)(2+5i)\)
  3. \(\displaystyle \dfrac{3-i}{4+5i}\)
  4. \(\displaystyle \dfrac{(2-1)^2}{(3+i)^2}\)
  5. \(\displaystyle 2+6i-(\overline{5+3i})\)
  6. \(\displaystyle (3+2i)(\overline{2-3i})\)
  7. \(\displaystyle (\overline{4-i})(\overline{1-4i})\)
3.

Represente no plano complexo os seguintes números complexos:

  1. \(\displaystyle z=3+2i\)
  2. \(\displaystyle z=4-i\)
  3. \(\displaystyle z=-2-2i\)
  4. \(\displaystyle z=-2+3i\)
4.

Escrevam na forma trigonométrica os seguintes números complexos:

  1. \(\displaystyle 6i\)
  2. \(\displaystyle 2+2i\)
  3. \(\displaystyle -8\sqrt{3}+8i\)
  4. \(\displaystyle 4\)
  5. \(\displaystyle 2-2i\)
  6. \(\displaystyle -3\)
  7. \(\displaystyle i\)
5.

Calcule a potência:

  1. \(\displaystyle (1+i\sqrt{3})^4\)
  2. \(\displaystyle (1-i\sqrt{3})^5\)
  3. \(\displaystyle (1-i)^{10}\)
6.

Determine o número complexo \(z\text{,}\) sabendo que \(w=10(\cos 20^o+i\sin 20^o)\) e \(zw=20\sqrt{3}(\cos 170^o+i\sin 170^o)\text{.}\)

7.

Calcule \(i^2, i^3, i^4, i^5\) e observe que as potências começam a se repetir depois de \(i^4\text{.}\) Comprove este fato mostrando que \(i^{4n+r}=i^r\) e aplique este resultado para calcular:

  1. \(\displaystyle i^{20}\)
  2. \(\displaystyle i^{72}\)
  3. \(\displaystyle i^{1041}\)
  4. \(\displaystyle 1+i+i^2+\cdots+i^{1992}\)
8.

Determine \(\alpha\) real para que \(\dfrac{2+\alpha i}{1-i}\) seja um imaginário puro.

9.

Resolva as equações:

  1. \(\displaystyle z+2\overline{z}=6+i\)
  2. \(\displaystyle (1+i)z+3i\overline{z}=2+i\)
  3. \(\displaystyle z^2=-5-12i\)
  4. \(\displaystyle z^2=i\)
  5. \(\displaystyle z^2=\overline{z}\)
  6. \(\displaystyle z^3=i\)
  7. \(\displaystyle z^4=-16\)
10.

Determine um polinômio de segundo grau, de coeficientes reais, que admite \(1-3i\) como raiz.
