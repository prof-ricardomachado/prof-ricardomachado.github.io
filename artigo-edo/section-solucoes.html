Seção 1 Pontos críticos e soluções.

Nesta seção é possível usar o SageMath para calcular pontos críticos de sistemas de EDOs (usando o método solve), obter soluções de sistemas lineares, tanto em casos gerais, como em casos particulares com condições iniciais (usando o método desolve_system). Também é possível obter soluções de EDOs (usando o método desolve).

Os códigos que estão disponíveis podem ser modificados e executados (apertando o botão "Evaluate (Sage)"). Estes códigos são enviados a um servidor e dentro de poucos segundos o resultado é retornado.

Tecnologia 1.1.

Pontos críticos para o sistema de EDOs:

\begin{equation} \begin{cases} \displaystyle\dfrac{dx}{dt} \amp \displaystyle= xa - \alpha xy = x(a-\alpha y)\\ \dfrac{dy}{dt} \amp \displaystyle = -cy + \gamma xy = y(-c+\gamma x) \end{cases}\label{sistema_pvp}\tag{1.1} \end{equation}

Os dados podem ser trocados para obter os pontos críticos de outros sistemas.

Os códigos no Sage são os seguintes:

Tecnologia 1.2.

Obtendo a solução da equação:

\begin{equation*} \frac{d}{dt} \left( \begin{array}{c} u \\ v \end{array}\right) = \left( \begin{array}{cc} 0 \amp \frac{-\alpha c }{\gamma} \\ \frac{\gamma a}{\alpha} \amp 0 \end{array}\right) \left( \begin{array}{c} u \\ v \end{array}\right). \end{equation*}
Tecnologia 1.3.

Obtendo a solução da equação:

\begin{equation*} \frac{d}{dt} \left( \begin{array}{c} u \\ v \end{array}\right) = \left( \begin{array}{cc} 0 \amp \frac{-\alpha c }{\gamma} \\ \frac{\gamma a}{\alpha} \amp 0 \end{array}\right) \left( \begin{array}{c} u \\ v \end{array}\right), \end{equation*}

com \(a = 1, \alpha = 0.5, c= 0.75, \gamma = 0.25\) e as condições iniciais \((x,y) = (2, 0.8)\text{.}\)
