Definição 1.1.11.

Uma permutação de uma lista de \(n\) elementos é chamada de permutação caótica, quando nenhum dos elementos da permutação está na posição original. O número de permutações caóticas de \(n\) elementos distintos é
\begin{equation*} D_n=n!\cdot\sum_{i=0}^{n}\dfrac{(-1)^i}{i!}. \end{equation*}
