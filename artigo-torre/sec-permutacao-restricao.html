Seção 2 Permutações com restrições

Nesta seção é possível usar o SageMath para calcular o número de permutações com diversos tipos de restições.

Os códigos que estão disponíveis podem ser modificados e executados (apertando o botão "Evaluate (Sage)"). Estes códigos são enviados a um servidor e dentro de poucos segundos o resultado é retornado.

Subseção 2.1 Permutações Caóticas

Definição 2.1.

Uma permutação caótica, ou desarranjo, dos elementos \(1, 2, \ldots, n\text{,}\) é uma permutação, na qual, o elemento \(1\) não pode ocupar a \(1\)ª posição, o elemento \(2\) não pode ocupar a \(2\)ª posição e assim por diante, até que o elemento \(n\) não possa ocupar a \(n\)ª posição. Por exemplo, para \(n=5\text{,}\) \((2,1,4,5,3)\) e \((5,4,1,2,3)\) são permutações caóticas.

Tecnologia 2.2.

Calculando o número de permutações caóticas através de polinômios de torre.

Tecnologia 2.4.

Função \(\verb|D(n)|\) para obter o número de permutações caóticas, usando o método \(\verb|rook_vector|\) para calcular os coeficientes dos polinômios de torre.

Tecnologia 2.5.

Função \(\verb|D(n)|\) para obter o número de permutações caóticas, usando o teorema:

Teorema 2.6.

A quantidade de permutações caóticas de \(n\) elementos é

\begin{equation*} D_n=\sum_{k=0}^{n}(-1)^k\dfrac{n!}{k!}. \end{equation*}

Subseção 2.2 Permutações com \(k\) Pontos Fixos

Corolário 2.7.

As permutações dos elementos \(1, 2, \ldots, n\text{,}\) na qual, \(k\) elementos estão em suas posições originais são chamadas de permutações com \(k\) pontos fixos. O número de permutações com \(k\) pontos fixos é denotado por \(D_{n,k}\) e pode ser calculado usando a expressão de \(D_n\) da seguinte maneira:

\begin{equation*} D_{n,k} = \binom{n}{k}\cdot D_{n-k}. \end{equation*}
Tecnologia 2.8.

Calculando o número de permutações \(k\)-posicionadas.

Função \(\verb|D(n,k)|\text{,}\) a função está predefinida com \(k=0\text{,}\) ou seja, se o valor de \(k\) não for informado, \(D_n\) será calculado.

Subseção 2.3 Permutações \(k\)-Posicionadas

Definição 2.9.

Uma permutação do conjunto \(\{1,2,\ldots, n\}\) é chamada de permutação \(k\)-posicionada se o \(k\)-ésimo elemento está na posição \(k\text{,}\) os elementos anteriores não estão em suas posições originais e os demais elementos podem ocupar qualquer posição. O número de permutações \(k\)-posicionadas é denotado por \(R_{n,k}\text{.}\) Por convenção, chamamos de permutação \((n+1)\)-posicionada quando nenhum elemento está em sua posição original.

Tecnologia 2.13.

Função \(\verb|R(n,k)|\text{,}\) para obter o número de permutações \(k\)-posicionadas, usando o teorema:

Teorema 2.14.

O número de permutações \(k\)-posicionadas de \(n\) elementos é dado por

\begin{equation*} R_{n,k} = \begin{cases} \displaystyle\sum_{i=0}^{k-1} (-1)^i\cdot(n-1-i)!\cdot \binom{k-1}{i} , \amp\text{ se } k\leq n;\\ D_{n}, \amp\text{ se k=n+1}. \end{cases} \end{equation*}

Subseção 2.4 Permutações Caóticas com Elementos Repetidos

Definição 2.15.

Uma permutação caótica com elementos repetidos é uma permutação na qual, os elementos nem podem ocupar suas posições originais, nem as posições dos seus elementos repetidos.
