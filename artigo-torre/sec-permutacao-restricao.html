Subseção 2.2 Permutações Caóticas

Tecnologia 2.3.

Calculando o número de permutações caóticas através de polinômios de torre.

Função \(\verb|D(n)|\text{,}\) para obter o número de permutações caóticas, usando o método \(\verb|rook_vector|\) para calcular os coeficientes dos polinômios de torre.

Função \(\verb|D(n)|\text{,}\) para obter o número de permutações caóticas, usando a expressão:

\begin{equation*} D_n=\sum_{k=0}^{n}(-1)^k\dfrac{n!}{k!}. \end{equation*}

Subseção 2.3 Permutações Quase Caóticas

Definição 2.4.

Chamaremos de permutação quase caótica, ou quase desarranjo, uma permutação na qual, pelo menos um elemento não podem ficar em sua posição original e no máximo o total de elementos menos um, não podem ficar em suas posições originais.

Tecnologia 2.5.

Calculando o número de permutações quase caóticas.

Função \(\verb|QD(n,m)|\text{,}\) para obter o número de permutações quase caóticas, usando o método \(\verb|rook_vector|\) para calcular os coeficientes dos polinômios de torre.

Função \(\verb|Q(n,m)|\text{,}\) para obter o número de permutações quase caóticas, usando a expressão:

\begin{equation*} QD_{n,m}= \sum_{k=0}^m (-1)^k\cdot(n-k)!\cdot \binom{m}{k}. \end{equation*}
