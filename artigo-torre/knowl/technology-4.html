Função \(\verb|D(n)|\text{,}\) para obter o número de permutações caóticas, usando o método \(\verb|rook_vector|\) para calcular os coeficientes dos polinômios de torre.

Função \(\verb|D(n)|\text{,}\) para obter o número de permutações caóticas, usando o teorema:

Teorema 2.3.

A quantidade de permutações caóticas de \(n\) elementos é

\begin{equation*} D_n=\sum_{k=0}^{n}(-1)^k\dfrac{n!}{k!}. \end{equation*}
