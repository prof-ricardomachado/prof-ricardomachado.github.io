Tecnologia 2.5.

Calculando o número de permutações \(k\) enfileiradas.

Função \(\verb|R(n,k)|\text{,}\) para obter o número de permutações \(k\) enfileiradas, usando o método \(\verb|rook_vector|\) para calcular os coeficientes dos polinômios de torre.

Função \(\verb|R(n,k)|\text{,}\) para obter o número de permutações quase caóticas, usando a expressão:

\begin{equation*} R_{n,k} = \begin{cases} \displaystyle\sum_{i=0}^{k-1} (-1)^i\cdot(n-1-i)!\cdot \binom{k-1}{i} , \amp\text{ se } k\leq n;\\ D_{n}, \amp\text{ se k=n+1}. \end{cases} \end{equation*}
