Tecnologia 2.5.

Calculando o número de permutações quase caóticas.

Função \(\verb|QD(n,m)|\text{,}\) para obter o número de permutações quase caóticas, usando o método \(\verb|rook_vector|\) para calcular os coeficientes dos polinômios de torre.

Função \(\verb|Q(n,m)|\text{,}\) para obter o número de permutações quase caóticas, usando a expressão:

\begin{equation*} QD_{n,m}= \sum_{k=0}^m (-1)^k\cdot(n-k)!\cdot \binom{m}{k}. \end{equation*}
