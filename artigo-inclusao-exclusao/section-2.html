Seção 2 O Princípio da Inclusão-Exclusão Generalizado

Definição 2.1.

Sejam \(\Omega\) um conjunto, \(A_1, A_2, \ldots, A_n\) subconjuntos de \(\Omega\text{.}\) Definimos os números \(S_0, S_1, S_2, \ldots, S_n\) da seguinte maneira:

\begin{align*} S_0=\amp~\#\Omega;\\ S_1=\amp~\sum_{i=1}^n\#A_i;\\ S_2=\amp~\sum_{1\leq i\lt j\leq n}^n\#(A_i\cap A_j);\\ \vdots\\ S_n=\amp~\#(A_1\cap A_2\cap \ldots\cap A_n). \end{align*}
Teorema 2.2.

Sejam \(A_1, A_2,\ldots,A_n\) subconjuntos de um conjunto \(\Omega\text{.}\)

  1. O número de elementos de \(\Omega\) que pertencem a exatamente \(p\) desses subconjuntos é dado por:
    \begin{align*} a_p \amp=~ C_p^0S_p - C_{p+1}^1S_{p+1}+C_{p+2}^2S_{p+2}+ \cdots +(-1)^nC_{n}^{n-p}S_{n}\\ \amp=~ \sum_{k=0}^{n-p}(-1)^kC_{p+k}^kS_{p+k}; \end{align*}
  2. O número de elementos de \(\Omega\) que pertencem a pelo menos \(p\) desses subconjuntos é dado por:
    \begin{align*} b_p \amp=~ C_{p-1}^0S_{p} - C_{p}^1S_{p+1}+C_{p+1}^2S_{p+2}+ \cdots +(-1)^nC_{n-1}^{n-p}S_{n}\\ \amp=~ \sum_{k=0}^{n-p}(-1)^kC_{p+k-1}^kS_{p+k}. \end{align*}
Tecnologia 2.3.

Escolha os valores dos campos, Vmin ,Vmax, lista e p, para determinar os valores de \(S_i\text{,}\) \(a_p\text{,}\) \(b_p\) e a cardinalidade do conjunto \(C\text{,}\) definido abaixo:

\begin{equation*} C = \{n\in \mathbb{N}~|~ n \text{ é múltiplo de } n_1, \text{ ou } n_2, \text{ ou } \ldots n_k \text{ e } Vmin \leq n\leq Vmax \}. \end{equation*}
