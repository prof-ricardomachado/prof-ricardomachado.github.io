Solução 3.6.4.4.1.
Seja \(F(x,y,z) = x^2 + y^2 - z^2 - xy = 0\text{.}\)
\(F_x = 2x - y\text{,}\) \(F_y = 2y - x\text{,}\) \(F_z = -2z\text{.}\)
\(\frac{\partial z}{\partial x} = - \frac{2x - y}{-2z} = \frac{2x - y}{2z}\text{.}\)
\(\frac{\partial z}{\partial y} = - \frac{2y - x}{-2z} = \frac{2y - x}{2z}\text{.}\)
