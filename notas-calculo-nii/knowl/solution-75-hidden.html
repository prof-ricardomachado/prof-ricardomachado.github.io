Para \((x,y) \neq (0,0)\text{,}\) a função é contínua por ser racional com denominador não nulo. Para garantir continuidade na origem, exigimos \(\lim_{(x,y) \to (0,0)} f(x,y) = L\text{.}\) Fatorando o numerador (diferença de quadrados):
\begin{equation*} x^4 - y^4 = (x^2 - y^2)(x^2 + y^2) \end{equation*}
Substituindo no limite:
\begin{equation*} \lim_{(x,y) \to (0,0)} \frac{(x^2 - y^2)(x^2 + y^2)}{x^2 + y^2} = \lim_{(x,y) \to (0,0)} (x^2 - y^2) = 0 - 0 = 0 \end{equation*}
Logo, para que a função seja contínua, devemos ter \(L = 0\).
