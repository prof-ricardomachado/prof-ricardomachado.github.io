Solução 3.5.3.1.

Usando a regra da cadeia do Caso I:
\begin{equation*} \frac{dh}{dt} = \frac{\partial f}{\partial x} \cdot \frac{dx}{dt} + \frac{\partial f}{\partial y} \cdot \frac{dy}{dt}\text{.} \end{equation*}
Calculando as derivadas:
  • \(\displaystyle \frac{\partial f}{\partial x} = 2xy + \frac{y^2}{xy^2} = 2xy + \frac{1}{x}\)
  • \(\displaystyle \frac{\partial f}{\partial y} = x^2 + \frac{2xy}{xy^2} = x^2 + \frac{2}{y}\)
  • \(\frac{dx}{dt} = 2t\) e \(\frac{dy}{dt} = 1\)
Substituindo \(x = t^2\) e \(y = t\text{:}\)
\begin{equation*} \frac{dh}{dt} = \left(2(t^2)(t) + \frac{1}{t^2}\right)(2t) + \left((t^2)^2 + \frac{2}{t}\right)(1) \end{equation*}
\begin{equation*} \frac{dh}{dt} = \left(2t^3 + \frac{1}{t^2}\right)2t + t^4 + \frac{2}{t} = 4t^4 + \frac{2}{t} + t^4 + \frac{2}{t} = 5t^4 + \frac{4}{t}\text{.} \end{equation*}
