Solução 2.2.3.2.1.

Igualamos a função a uma constante \(k\) (note que a imagem da exponencial é sempre positiva, então \(k > 0\)):
\begin{equation*} e^{xy} = k \implies xy = \ln(k) \end{equation*}
Para \(k \neq 1\text{,}\) temos que \(\ln(k)\) é uma constante diferente de zero. A equação \(xy = c\) (onde \(c = \ln(k)\)) representa uma família de hipérboles.
Especificamente quando \(k = 1\text{:}\)
\begin{equation*} xy = \ln(1) \implies xy = 0 \end{equation*}
Para que o produto seja zero, devemos ter \(x = 0\) ou \(y = 0\text{.}\) Portanto, a curva de nível \(k=1\) é a união dos eixos coordenados (o eixo \(x\) e o eixo \(y\) inteiros).
