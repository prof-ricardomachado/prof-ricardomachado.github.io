Solução 3.4.5.7.1.
Área \(A = xy\text{.}\) Diferencial \(dA = y \, dx + x \, dy\text{.}\)
Valores: \(x = 1200\text{,}\) \(y = 1800\text{.}\) Erros: \(dx = 10\text{,}\) \(dy = 15\) (consideramos positivos para obter o erro máximo).
\(dA = 1800(10) + 1200(15) = 18000 + 18000 = 36000\text{ m}^2\text{.}\)
