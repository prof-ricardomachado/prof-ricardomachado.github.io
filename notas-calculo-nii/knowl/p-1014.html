Agora, derivamos novamente para obter as de 2ª ordem:
  • \(\displaystyle \frac{\partial^2 f}{\partial x^2} = \frac{\partial}{\partial x}(3x^2y + 2xy^4) = 6xy + 2y^4\)
  • \(\displaystyle \frac{\partial^2 f}{\partial y \partial x} = \frac{\partial}{\partial y}(3x^2y + 2xy^4) = 3x^2 + 8xy^3\)
  • \(\displaystyle \frac{\partial^2 f}{\partial x \partial y} = \frac{\partial}{\partial x}(x^3 + 4x^2y^3) = 3x^2 + 8xy^3\)
  • \(\displaystyle \frac{\partial^2 f}{\partial y^2} = \frac{\partial}{\partial y}(x^3 + 4x^2y^3) = 12x^2y^2\)
