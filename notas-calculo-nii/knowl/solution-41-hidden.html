Igualamos a função à constante \(k\text{:}\)
\begin{equation*} x^2 + y^2 = k \end{equation*}
Analisando para os valores dados:
  • Para \(k = 0\text{:}\) \(x^2 + y^2 = 0\text{.}\) A única solução real é o ponto (origem) \((0,0)\text{.}\)
  • Para \(k = 1\text{:}\) \(x^2 + y^2 = 1\text{.}\) É uma circunferência centrada na origem com raio 1.
  • Para \(k = 4\text{:}\) \(x^2 + y^2 = 4\text{.}\) É uma circunferência centrada na origem com raio 2.
  • Para \(k = 9\text{:}\) \(x^2 + y^2 = 9\text{.}\) É uma circunferência centrada na origem com raio 3.
O mapa de contorno é formado por circunferências concêntricas. Quanto maior o raio, maior a "altura" (\(z\)) da superfície.
