Fazendo \(2x + 4y = k\text{,}\) obtemos a equação da reta \(y = -\frac{1}{2}x + \frac{k}{4}\text{.}\)
  • Para \(k = -4\text{:}\) \(y = -\frac{1}{2}x - 1\text{.}\)
  • Para \(k = 0\text{:}\) \(y = -\frac{1}{2}x\) (reta pela origem).
  • Para \(k = 4\text{:}\) \(y = -\frac{1}{2}x + 1\text{.}\)
As curvas de nível são retas paralelas, indicando que a superfície é um plano inclinado.
