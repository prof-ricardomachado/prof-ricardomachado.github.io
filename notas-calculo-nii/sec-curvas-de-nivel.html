    Seção 2.2 Curvas de Nível

    Representar graficamente uma função de duas variáveis \(z = f(x,y)\) exige o desenho de uma superfície no espaço tridimensional (\(\mathbb{R}^3\)), o que nem sempre é uma tarefa simples. Para contornar essa dificuldade e compreender o comportamento da função usando apenas o plano bidimensional (\(\mathbb{R}^2\)), utilizamos a técnica das curvas de nível.

    Subseção 2.2.1 O que são Curvas de Nível?

    Dada uma função \(z = f(x,y)\) e uma constante \(k\) pertencente à imagem da função, a curva de nível \(k\) é o conjunto de todos os pontos \((x,y)\) no domínio de \(f\) que satisfazem a equação:
    \begin{equation*} f(x,y) = k \end{equation*}
    Geometricamente, fazer \(f(x,y) = k\) equivale a fatiar a superfície 3D com um plano horizontal na altura \(z = k\text{.}\) A curva resultante dessa interseção é então projetada no plano \(xy\) (o "chão"). O mapa formado por várias dessas curvas para diferentes valores de \(k\) é chamado de mapa de contorno.

    Aplicações Práticas.

    • Topografia: Mapas topográficos usam curvas de nível para representar a altitude (relevo) de montanhas e vales em um mapa plano.
    • Meteorologia: As isotermas são curvas de nível que unem pontos de mesma temperatura; as isóbaras unem pontos de mesma pressão atmosférica.
    • Eletromagnetismo: As curvas equipotenciais unem pontos do espaço que possuem o mesmo potencial elétrico.

    Subseção 2.2.2 Exemplos Práticos

    Exemplo 2.2.1. Curvas de Nível de um Paraboloide.

    Descreva e esboce as curvas de nível da função \(f(x,y) = x^2 + y^2\) para \(k = 0, 1, 4, 9\text{.}\)
    Solução.
    Igualamos a função à constante \(k\text{:}\)
    \begin{equation*} x^2 + y^2 = k \end{equation*}
    Analisando para os valores dados:
    • Para \(k = 0\text{:}\) \(x^2 + y^2 = 0\text{.}\) A única solução real é o ponto (origem) \((0,0)\text{.}\)
    • Para \(k = 1\text{:}\) \(x^2 + y^2 = 1\text{.}\) É uma circunferência centrada na origem com raio 1.
    • Para \(k = 4\text{:}\) \(x^2 + y^2 = 4\text{.}\) É uma circunferência centrada na origem com raio 2.
    • Para \(k = 9\text{:}\) \(x^2 + y^2 = 9\text{.}\) É uma circunferência centrada na origem com raio 3.
    O mapa de contorno é formado por circunferências concêntricas. Quanto maior o raio, maior a "altura" (\(z\)) da superfície.

    Exemplo 2.2.2. Curvas de Nível de um Plano.

    Encontre a equação das curvas de nível para a função linear \(g(x,y) = 3x - 2y\text{.}\)
    Solução.
    Igualamos a expressão a uma constante genérica \(k\text{:}\)
    \begin{equation*} 3x - 2y = k \implies 2y = 3x - k \implies y = \frac{3}{2}x - \frac{k}{2} \end{equation*}
    Para qualquer valor de \(k \in \mathbb{R}\text{,}\) a equação representa uma reta com coeficiente angular \(m = 3/2\text{.}\) Portanto, as curvas de nível são uma família de retas paralelas entre si. Superfícies planas sempre produzem curvas de nível que são retas paralelas.

    Exercícios 2.2.3 Exercícios Propostos

    1.

    Determine e descreva geometricamente as curvas de nível da função \(h(x,y) = \sqrt{100 - x^2 - y^2}\) para \(k = 0, 6\) e \(10\text{.}\)
    Solução.
    Igualamos a função a \(k\text{:}\)
    \begin{equation*} \sqrt{100 - x^2 - y^2} = k \implies 100 - x^2 - y^2 = k^2 \implies x^2 + y^2 = 100 - k^2 \end{equation*}
    Substituindo os valores de \(k\text{:}\)
    • Para \(k = 0\text{:}\) \(x^2 + y^2 = 100\) (Circunferência de raio 10).
    • Para \(k = 6\text{:}\) \(x^2 + y^2 = 100 - 36 \implies x^2 + y^2 = 64\) (Circunferência de raio 8).
    • Para \(k = 10\text{:}\) \(x^2 + y^2 = 100 - 100 \implies x^2 + y^2 = 0\) (Apenas o ponto \((0,0)\)).
    A superfície geométrica é um hemisfério superior. As curvas de nível são circunferências concêntricas que diminuem de raio à medida que a altura (\(k\)) aumenta.

    2.

    Descreva a família de curvas de nível da função \(f(x,y) = e^{xy}\text{.}\) O que acontece especificamente quando \(k = 1\text{?}\)
    Solução.
    Igualamos a função a uma constante \(k\) (note que a imagem da exponencial é sempre positiva, então \(k > 0\)):
    \begin{equation*} e^{xy} = k \implies xy = \ln(k) \end{equation*}
    Para \(k \neq 1\text{,}\) temos que \(\ln(k)\) é uma constante diferente de zero. A equação \(xy = c\) (onde \(c = \ln(k)\)) representa uma família de hipérboles.
    Especificamente quando \(k = 1\text{:}\)
    \begin{equation*} xy = \ln(1) \implies xy = 0 \end{equation*}
    Para que o produto seja zero, devemos ter \(x = 0\) ou \(y = 0\text{.}\) Portanto, a curva de nível \(k=1\) é a união dos eixos coordenados (o eixo \(x\) e o eixo \(y\) inteiros).
