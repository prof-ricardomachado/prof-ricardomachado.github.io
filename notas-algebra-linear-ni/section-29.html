Seção 4.12 Exercícios

Exercícios 4.12.1 Exercícios

1.
Sejam \(U\) e \(V\) subespaços vetoriais de um espaço vetorial \(W\text{,}\) demonstre que \(\dim (U + V) = \dim U + \dim V - \dim (U \cap V).\)
2.
Sejam \(P_1\) e \(P_2\) dois planos no \(\mathbb{R}^3\) passando pela origem (isto é, dois subespaços de dimensão \(2\)). Determine \(\dim (P_1 \cap P_2)\) quando:
  1. \(P_1 + P_2 = \mathbb{R}^3\text{.}\)
  2. \(P_1 + P_2 \neq \mathbb{R}^3\text{.}\)
3.
Determine o vetor coordenada \([v]_{\alpha}\) de \(v=(6,2)\) em relação à base \(\alpha\) nos seguintes casos:
  1. \(\alpha = \{(3,0), (0,2)\}\text{.}\)
  2. \(\alpha = \{(0,1), (1,0)\}\text{.}\)
  3. \(\alpha = \{(1,2), (2,1)\}\text{.}\)
4.
Considere a base \(\beta = \{(1,0,0), (0,1,0), (1,-1,1)\}\) do espaço vetorial euclidiano \(\mathbb{R}^3\text{.}\) Determine o vetor coordenada de \(v\) em relação à base \(\beta\text{,}\) onde:
  1. \(v = (2,-3,4)\text{.}\)
  2. \(v = (3,5,6)\text{.}\)
  3. \(v = (1,-1,1)\text{.}\)
5.
Seja \(\beta = \{3, 2x, -x^2\}\) uma base de \(\mathcal{P}_2[x](\mathbb{R})\text{.}\) Determine o vetor coordenada de \(v = 6 - 4x - 3x^2\) em relação à base \(\beta\text{.}\)
6.
Sejam os vetores \(v_1 = (1,0,-1)\text{,}\) \(v_2 = (1,2,1)\) e \(v_3 = (0,-1,0)\) do espaço euclidiano \(\mathbb{R}^3\text{.}\)
  1. Mostre que \(\beta = \{v_1, v_2, v_3\}\) é base de \(\mathbb{R}^3\text{.}\)
  2. Escreva \(e_1 = (1,0,0)\text{,}\) \(e_2 = (0,1,0)\) e \(e_3 = (0,0,1)\) como combinação linear dos vetores da base \(\beta\text{.}\)
7.
Seja \(\beta = \{1-x, x-x^2, 2+x^2\}\) uma base de \(\mathcal{P}_2[x](\mathbb{R})\text{.}\)
  1. Mostre que \(\beta\) é uma base de \(\mathcal{P}_2[x](\mathbb{R})\text{.}\)
  2. Encontre as coordenadas de \(p(x) = 7 - x + 2x^2\) em relação à base \(\beta\text{.}\)
  3. Encontre a matriz de mudança da base \(\beta\) para a base canônica \(\mathcal{C} = \{1, x, x^2\}\text{.}\)
8.
Determine as coordenadas do vetor \(v = \begin{bmatrix} 2 \amp 5 \\ -8 \amp 7 \end{bmatrix} \in \mathcal{M}_{2 \times 2}(\mathbb{R})\) em relação às seguintes bases de \(\mathcal{M}_{2 \times 2}(\mathbb{R})\text{:}\)
  1. \(\displaystyle \mathcal{C} = \left\{ \begin{bmatrix} 1 \amp 0 \\ 0 \amp 0 \end{bmatrix}, \begin{bmatrix} 0 \amp 1 \\ 0 \amp 0 \end{bmatrix}, \begin{bmatrix} 0 \amp 0 \\ 1 \amp 0 \end{bmatrix}, \begin{bmatrix} 0 \amp 0 \\ 0 \amp 1 \end{bmatrix} \right\}\)
  2. \(\displaystyle \beta = \left\{ \begin{bmatrix} 1 \amp 0 \\ 0 \amp 0 \end{bmatrix}, \begin{bmatrix} 1 \amp 1 \\ 0 \amp 0 \end{bmatrix}, \begin{bmatrix} 1 \amp 1 \\ 1 \amp 0 \end{bmatrix}, \begin{bmatrix} 1 \amp 1 \\ 1 \amp 1 \end{bmatrix} \right\}\)
9.
Sejam \(\mathcal{C} = \{(1,0), (0,1)\}\text{,}\) \(\beta_1 = \{(-1,1), (1,1)\}\text{,}\) \(\beta_2 = \{(\sqrt{3},1), (\sqrt{3},-1)\}\text{.}\)
    1. \([I]_{\mathcal{C}}^{\beta_1}\text{.}\)
    2. \([I]_{\beta_1}^{\mathcal{C}}\text{.}\)
    3. \([I]_{\beta_1}^{\beta_2}\text{.}\)

  2. Quais são as coordenadas do vetor \(v = (3,-2)\) em relação à base:

    1. \(\mathcal{C}\text{.}\)
    2. \(\beta_1\text{.}\)
    3. \(\beta_2\text{.}\)
  3. As coordenadas de um vetor \(v\) em relação à base \(\mathcal{C}\) são dadas por
    \begin{equation*} [v]_{\mathcal{C}} = \begin{bmatrix} 4 \\ 0 \end{bmatrix}\text{.} \end{equation*}
    Quais são as coordenadas de \(v\) em relação às bases \(\beta_1\) e \(\beta_2\text{.}\)
10.
Seja \(V\) o espaço vetorial das matrizes \(2 \times 2\) triangulares inferiores. Considere as seguintes bases:
\begin{equation*} \alpha = \left\{ \begin{bmatrix} 1 \amp 0 \\ 0 \amp 0 \end{bmatrix}, \begin{bmatrix} 0 \amp 0 \\ 1 \amp 0 \end{bmatrix}, \begin{bmatrix} 0 \amp 0 \\ 0 \amp 1 \end{bmatrix} \right\}\text{,} \end{equation*}
\begin{equation*} \beta = \left\{ \begin{bmatrix} 1 \amp 0 \\ 3 \amp 0 \end{bmatrix}, \begin{bmatrix} 0 \amp 0 \\ 1 \amp 0 \end{bmatrix}, \begin{bmatrix} 1 \amp 0 \\ 1 \amp 2 \end{bmatrix} \right\}\text{.} \end{equation*}
Determine as matrizes de mudança de base \([I]_{\beta}^{\alpha}\) e \([I]_{\alpha}^{\beta}\text{.}\)
