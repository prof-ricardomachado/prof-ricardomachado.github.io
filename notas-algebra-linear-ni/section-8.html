Seção 2.4 A Forma Escada

Definição 2.4.1.
Uma matriz é dita ter sido linha reduzida à forma escada se
  1. O primeiro elemento não nulo (chamado às vezes de pivô) de uma linha não nula é 1.
  2. Cada coluna que contém o pivô de alguma linha tem todos os seus outros elementos iguais a zero.
  3. Toda linha nula ocorre abaixo de todas as linhas não nulas
  4. O pivô de uma linha não nula está sempre estritamente à direita do pivô da linha acima dele.
Observação 2.4.2.
Pode-se mostrar que toda matriz \(A_{m\times n}\) é linha equivalente a uma única matriz-linha reduzida à forma escada.
Exemplo 2.4.3.
Determine se as seguintes matrizes abaixo se encontram linha reduzidas à forma escada:
  1. \(\displaystyle \begin{bmatrix} 1 \amp 0 \amp 0 \amp 0\\ 0 \amp 1 \amp -1 \amp 0 \\ 0 \amp 0 \amp 1 \amp 0 \end{bmatrix}\)
  2. \(\displaystyle \begin{bmatrix} 0 \amp 2 \amp 1 \\ 1 \amp 0 \amp -3 \\ 0 \amp 0 \amp 0 \end{bmatrix}\)
  3. \(\displaystyle \begin{bmatrix} 0 \amp 1 \amp -3 \amp 0 \amp 1\\ 0 \amp 0 \amp 0 \amp 0 \amp 0 \\ 0 \amp 0 \amp 0 \amp -1 \amp 2 \end{bmatrix}\)
  4. \(\displaystyle \begin{bmatrix} 0 \amp 1 \amp -3 \amp 0 \amp 2\\ 0 \amp 0 \amp 0 \amp 1 \amp 2 \\ 0 \amp 0 \amp 0 \amp 0 \amp 0 \end{bmatrix}\)
Solução

Verifiquemos se as matrizes abaixo cumprem as condições (a) até (d):

  1. \(\displaystyle \begin{bmatrix} 1 \amp 0 \amp 0 \amp 0\\ 0 \amp 1 \amp -1 \amp 0 \\ 0 \amp 0 \amp 1 \amp 0 \end{bmatrix} \quad\text{Não, pois não cumpre (b).}\)
  2. \(\displaystyle \begin{bmatrix} 0 \amp 2 \amp 1 \\ 1 \amp 0 \amp -3 \\ 0 \amp 0 \amp 0 \end{bmatrix} \quad\text{Não, pois não cumpre (a) e (d).}\)
  3. \(\displaystyle \begin{bmatrix} 0 \amp 1 \amp -3 \amp 0 \amp 1\\ 0 \amp 0 \amp 0 \amp 0 \amp 0 \\ 0 \amp 0 \amp 0 \amp -1 \amp 2 \end{bmatrix} \quad\text{Não, pois não cumpre (a) e (c).}\)
  4. \(\displaystyle \begin{bmatrix} 0 \amp 1 \amp -3 \amp 0 \amp 2\\ 0 \amp 0 \amp 0 \amp 1 \amp 2 \\ 0 \amp 0 \amp 0 \amp 0 \amp 0 \end{bmatrix} \quad\text{Sim, pois cumpre todas as condições.}\)
