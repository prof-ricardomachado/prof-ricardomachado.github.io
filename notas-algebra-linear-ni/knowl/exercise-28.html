Exercício 20.

Seja \(A\) uma matriz \(n\times n.\) Demonstre as seguintes afirmações:
  1. Se \(A\) é inversível e existe uma matriz \(B,\,n\times n,\) tal que \(AB=\underline{0},\) então \(B=\underline{0}.\)
  2. Se \(A\) não é inversível, existe uma matriz não nula \(B,\,n\times n\) tal que \(AB=\underline{0}.\)
