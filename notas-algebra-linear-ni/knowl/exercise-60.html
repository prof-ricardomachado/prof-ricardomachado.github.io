Exercício 19.

Definimos \(W_1\oplus W_2 = \{ w_1 + w_2 \mid w_1 \in W_1,\, w_2 \in W_2,\, W_1 \cap W_2 = \{\underline{0}\} \}\) como a soma direta dos subespaços \(W_1\) e \(W_2\text{.}\) Se

\(W_1=\{(x,\,y,\,z,\,t)\in \mathbb{R}^4 \mid x+y=0,\, z-t=0\}\) e \(W_2=\{(x,\,y,\,z,\,t)\in \mathbb{R}^4 \mid x-y-z=0\}\text{.}\)
  1. Exiba uma base para \(W_1\) e \(W_2\text{.}\)
  2. Exiba uma base para \(W_1 \cap W_2\text{.}\)
  3. Exiba uma base para \(W_1 + W_2\text{.}\)
  4. \(W_1 + W_2\) é soma direta?
  5. \(W_1 + W_2 = \mathbb{R}^4\text{?}\)
