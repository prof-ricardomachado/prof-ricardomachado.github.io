Exercício 5.

Determine se os seguintes conjuntos são espaços vetoriais sobre \(\mathbb{R}\text{:}\)

  1. \(\displaystyle S = \{(x_1,x_2,x_3,x_4) \in \mathbb{R}^4 \,;\, x_1 = 2x_2, x_2 + 4x_4 = 0, 4x_2 + x_3 = 0\}\)
  2. \(\displaystyle S = \{ [a_{ij}] \in \mathcal{M}_{2\times 3}(\mathbb{R}) \,;\, a_{21}+a_{23}=0, a_{11}=0, a_{12}+a_{22}+1=0 \}\)
  3. \(\displaystyle S = \{ ax^3 + bx^2 + cx + d \in \mathcal{P}_3[x](\mathbb{R}) \,;\, a+b+c+d=0, b=2c, a=3d \}\)
  4. \(\displaystyle S \subset \mathcal{M}_{n\times n}(\mathbb{R})\text{, onde } S \text{ é o conjunto das matrizes simétricas}\)
  5. O conjunto dos vetores de \(\mathbb{R}^n\) cujas coordenadas formam uma progressão aritmética.
  6. O conjunto dos vetores de \(\mathbb{R}^n\) cujas primeiras \(k\) coordenadas são iguais.
  7. O conjunto dos vetores de \(\mathbb{R}^n\) que têm \(k\) coordenadas iguais.
  8. O conjunto das retas no espaço euclidiano \(\mathbb{R}^3\text{.}\)
  9. O conjunto das retas que passam na origem no espaço euclidiano \(\mathbb{R}^3\text{.}\)
