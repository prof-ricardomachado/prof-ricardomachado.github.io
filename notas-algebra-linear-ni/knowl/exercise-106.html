Exercício 16.

Seja \(V=\mathcal{M}_{2\times 2}(\mathbb{R})\) e considere a base canônica deste espaço descrita por
\begin{equation*} \mathcal{C}=\left\{ \left( \begin{array}{rr} 1 \amp 0 \\ 0 \amp 0 \end{array} \right), \left( \begin{array}{rr} 0 \amp 1 \\ 0 \amp 0 \end{array} \right), \left( \begin{array}{rr} 0 \amp 0 \\ 1 \amp 0 \end{array} \right), \left( \begin{array}{rr} 0 \amp 0 \\ 0 \amp 1 \end{array} \right) \right\}. \end{equation*}
Seja \(T: V\to \mathbb{R}^2\) descrita por
\begin{equation*} T\left( \begin{array}{rr} a \amp b \\ c \amp d \end{array} \right)= (a+d, b+c). \end{equation*}
  1. (i) Encontre \([T]_{\mathcal{C}_2}^{\mathcal{C}}\text{,}\) onde \(\mathcal{C}_2\) é a base canônica do espaço \(\mathbb{R}^2\text{.}\)
  2. (ii) Se \(S: \mathbb{R}^2 \to V\) é tal que \([S]_\mathcal{C}^{\mathcal{C}_2}=\left( \begin{array}{rr} 2 \amp 1 \\ 1 \amp -1 \\ -1 \amp 0 \\ 0 \amp 1 \end{array} \right)\text{,}\) determine \(S\) explicitamente e, se possível, \((a, b)\) tal que \(S(a, b)=\left( \begin{array}{rr} 1 \amp 0 \\ 0 \amp 1 \end{array} \right)\text{.}\)
