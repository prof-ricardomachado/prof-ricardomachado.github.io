A matriz dos coeficientes do sistema é
\begin{align*} \mathbf{A} = \begin{bmatrix} 2 \amp -3 \amp 7 \\ 1 \amp 0 \amp 3 \\ 0 \amp 2 \amp -1 \end{bmatrix} \end{align*}
O determinante de \(\mathbf{A}\) é
\begin{align*} \det \begin{bmatrix} 2 \amp -3 \amp 7 \\ 1 \amp 0 \amp 3 \\ 0 \amp 2 \amp -1 \end{bmatrix} = -1 \neq 0 \end{align*}
Logo, o sistema admite uma única solução e podemos usar a Regra de Cramer.
Assim, sendo \(\mathbf{b} = [1 \ 5 \ 0]^T\) o vetor de termos independentes associado ao sistema, temos
\begin{align*} x =\amp \frac{ \begin{vmatrix} 1 \amp -3 \amp 7 \\ 5 \amp 0 \amp 3 \\ 0 \amp 2 \amp -1 \end{vmatrix} }{-1} = -49\\ y =\amp \frac{ \begin{vmatrix} 2 \amp 1 \amp 7 \\ 1 \amp 5 \amp 3 \\ 0 \amp 0 \amp -1 \end{vmatrix} }{-1} = 9\\ z =\amp \frac{ \begin{vmatrix} 2 \amp -3 \amp 1 \\ 1 \amp 0 \amp 5 \\ 0 \amp 2 \amp 0 \end{vmatrix} }{-1} = 18 \end{align*}
Assim, o sistema possui única solução \((x,y,z) = (-49, 9, 18)\text{.}\)
