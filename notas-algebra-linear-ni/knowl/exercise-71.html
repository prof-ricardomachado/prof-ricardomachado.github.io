Exercício 10.

Seja \(V\) o espaço vetorial das matrizes \(2 \times 2\) triangulares inferiores. Considere as seguintes bases:
\begin{equation*} \alpha = \left\{ \begin{bmatrix} 1 \amp 0 \\ 0 \amp 0 \end{bmatrix}, \begin{bmatrix} 0 \amp 0 \\ 1 \amp 0 \end{bmatrix}, \begin{bmatrix} 0 \amp 0 \\ 0 \amp 1 \end{bmatrix} \right\}\text{,} \end{equation*}
\begin{equation*} \beta = \left\{ \begin{bmatrix} 1 \amp 0 \\ 3 \amp 0 \end{bmatrix}, \begin{bmatrix} 0 \amp 0 \\ 1 \amp 0 \end{bmatrix}, \begin{bmatrix} 1 \amp 0 \\ 1 \amp 2 \end{bmatrix} \right\}\text{.} \end{equation*}
Determine as matrizes de mudança de base \([I]_{\beta}^{\alpha}\) e \([I]_{\alpha}^{\beta}\text{.}\)
