Definição 2.4.1.

Uma matriz é dita ter sido linha reduzida à forma escada se
  1. O primeiro elemento não nulo (chamado às vezes de pivô) de uma linha não nula é 1.
  2. Cada coluna que contém o pivô de alguma linha tem todos os seus outros elementos iguais a zero.
  3. Toda linha nula ocorre abaixo de todas as linhas não nulas
  4. O pivô de uma linha não nula está sempre estritamente à direita do pivô da linha acima dele.
