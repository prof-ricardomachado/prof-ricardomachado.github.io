Definição 2.6.1.

(Soluções de um Sistema Linear) Considere um sistema de equações lineares da forma \(Ax = b\text{.}\) Então, temos três casos possíveis:
  1. O sistema possui uma única solução. Neste caso, diz-se que o sistema é possível e determinado.
  2. O sistema possui infinitas soluções. Neste caso, diz-se que o sistema é possível e indeterminado.
  3. O sistema não possui solução. Neste caso, diz-se que o sistema é impossível.
